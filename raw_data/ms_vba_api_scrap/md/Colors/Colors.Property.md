# Colors Property

## Business Description
Returns or sets colors in the palette for a Chart object. The palette has 56 entries, each represented by an RGB value. Read/write Variant.

## Behavior
Returns or sets colors in the palette for a Chart object. The palette has 56 entries, each represented by an RGB value. Read/write Variant.

## Example Usage
```vba
ActiveChart.Colors(5) = RGB(255, 0, 0)
```
