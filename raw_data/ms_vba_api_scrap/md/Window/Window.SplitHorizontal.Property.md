# Window SplitHorizontal Property

## Business Description
Returns or sets the location of the horizontal window split, in points. Read/write Double.

## Behavior
Returns or sets the location of the horizontal window split, in points. Read/writeDouble.

## Example Usage
```vba
Workbooks("BOOK1.XLS").Worksheets("Sheet1").Activate 
ActiveWindow.SplitHorizontal= 216
```
