# Name Object

## Business Description
Represents a defined name for a range of cells. Names can be either built-in names — such as Database, Print_Area, and Auto_Open — or custom names.

## Behavior
Represents a defined name for a range of cells. Names can be either built-in names — such as Database, Print_Area, and Auto_Open — or custom names.

## Example Usage
```vba
ActiveWorkbook.Names("mySortRange").Delete
```
